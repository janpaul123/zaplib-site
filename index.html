Zaplib - making web apps fast
Creators need an immediate connection to what they're creating. — Bret Victor, Inventing on Principle
We love fast tools. Unfortunately, many tools are built on the web, and the web wasn’t built for performance. This isn’t a problem for static sites or thin clients. But for highly-interactive tools, JavaScript just doesn’t cut it.
Rust is fast by default — it lets you write performance-intensive code naturally. And, it now runs on the web! However, it’s early days; there are limited frameworks and libraries.
Zaplib is an open-source Rust WebAssembly framework. It allows you to rewrite pieces of your app in Rust and have them seamlessly run within your existing JS app.
Every profession deserves a fast tool. We hope to empower the next generation of app developers to empower the next generation of engineers, architects, designers, policy makers, scientists, creatives, artists...
